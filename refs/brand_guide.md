Brand Style: Aerospace / Mission-Driven Minimalism
1. Color System
Core Palette (authoritative, low-noise)
Background
Deep Space Black — #070A0F
Primary background. Slightly blue-tinted black reads more premium than pure black.
Orbital Black — #0C111A
Used for sections, nav bars, subtle layering.
Text
Primary White — #F5F7FA
Headlines and key body text.
Muted White — #B8C0CC
Secondary copy, labels, metadata.
Accent (used sparingly)
Earth Blue — #2F80ED
CTAs, links, active states.
Atmosphere Blue — #56CCF2
Hover states, subtle glows, highlights.
Utility / Dividers
Starlight Gray — #1B2230
Hairline dividers, card borders.
Dust Gray — #8892A6
Disabled text, subtle UI hints.
Rules
Accents should be <5% of the UI
No gradients on text
No more than one accent color per screen
2. Typography
Primary Font
Inter
Weights used: 400, 500, 600
Reason: neutral, modern, extremely readable, trusted in enterprise + space tech
Alternative (slightly more “aerospace”)
Space Grotesk
Use only for headlines if you want more character
Pair with Inter for body text
Do not use
Serif fonts
Rounded “friendly” fonts
Display fonts or condensed styles
